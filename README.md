## 🦆 lfetch
Uma ferramenta de busca de sistema mínima e rápida para Windows escrita em C puro usando WinAPI.
## ⚡ sobre
O `winfetch` é uma alternativa leve a ferramentas como o neofetch, criado do zero usando APIs de baixo nível do Windows.

Sem dependências externas.
Apenas informações do sistema e um pato.
